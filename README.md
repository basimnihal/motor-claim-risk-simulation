
# 📊 Motor Insurance Claim Simulation Model (Excel)

## 📌 Project Overview

This project implements a Frequency–Severity based insurance risk simulation model in Excel.

The objective is to simulate claim behavior across a portfolio of policies and validate the model by comparing expected loss with simulated actual loss.

---

## 🧮 Model Structure

The model follows a classical actuarial framework:

1. Claim Frequency Simulation  
   - Discrete random claim count per policy  

2. Claim Severity Simulation  
   - Continuous random claim size distribution  

3. Total Loss Calculation  
   - Total Loss = Frequency × Severity  

4. Model Validation  
   - Expected Loss per Policy  
   - Simulated Actual Average Loss  
   - Loss Ratio Comparison  

---

## 📊 Key Outputs

- Number of Policies: 100  
- Average Claim Frequency  
- Average Claim Severity  
- Expected Loss per Policy  
- Actual Average Loss  
- Loss Ratio (Actual / Expected)

The results show close alignment between expected and simulated losses, demonstrating statistical consistency.
